# My Webtoon Collection(미완성)
### 설명
내가 즐겨보는 웹툰을 플랫폼에 상관없이 한 곳에 모아 보자!  
웹툰을 앱에 등록해두면, 새로운 편이 업데이트 될 때마다 알림으로 알려 줘요!
